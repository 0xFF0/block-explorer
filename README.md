## Dependencies

[Meteor](https://www.meteor.com/install)

## Install

	git clone -b dev https://github.com/theQRL/block-explorer.git
	cd block-explorer
	meteor npm install
	meteor npm install --save meteor-babel-helpers

## Run

	meteor
